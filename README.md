# Vending_Machine

## Technologies used:
1. Backend: NodeJs with TypeScript, ExpressJs
2. Frontend: ReactJs
3. Database: MongoDB

## Steps to run project:

1. Backend:

cd backend
nano .env (put the values in same format as .env.example)
npm install
npm start

To run as docker container, use Dockerfile

2. Backend:

cd frontend \n
npm install
npm start

To run as docker container, use Dockerfile

3. Database:

cd mongodb
docker-compose up -d
